# 开头钩子与标题公式

## 抖音爆款标题公式

### 公式库

| 公式 | 结构 | 示例 |
|------|------|------|
| 1 | 反常识 + 具体数字 | 《你手机里有 100 万倍于登月计算机的能力，但...》 |
| 2 | 悬念 + 悬念升级 | 《这家公司每年亏 100 亿，却没人说它不行》 |
| 3 | 生活场景 + 技术揭秘 | 《为什么刷短视频停不下来？背后的算法太狠了》 |
| 4 | 权威 + 颠覆认知 | 《MIT 研究：AI 正在学会"说谎"》 |
| 5 | 时间紧迫感 | 《趁还没普及，先搞懂什么是量子计算》 |
| 6 | 对比反差 | 《同样的 AI，有人效率翻倍，有人被裁员》 |
| 7 | "你以为...其实" | 《你以为 AI 在思考，其实它在猜》 |
| 8 | 数字 + 利益点 | 《3 分钟看懂区块链，以后跟朋友吹牛够用》 |

### 标题检查清单

- [ ] 15-25 字（最佳区间）
- [ ] 包含数字或对比（提升点击率）
- [ ] 制造好奇心缺口
- [ ] 不标题党（内容能接住承诺）
- [ ] 读完标题就想点进去

## 开头钩子库

### 类型 1：反常识型（最常用，效果最好）

```
❌ 普通开头："今天给大家讲讲什么是大语言模型。"
✅ 反常识开头："大语言模型其实不'理解'任何东西。它只是概率的高手。"
```

### 类型 2：数据冲击型

```
❌ 普通开头："AI 发展很快。"
✅ 数据冲击："过去 2 年，AI 的能力翻了一百倍。而人类智力，基本没变。"
```

### 类型 3：故事型

```
❌ 普通开头："马斯克创办了很多公司。"
✅ 故事型："2008 年，马斯克站在火箭残骸旁边哭了。SpaceX 第三次发射失败，
公司只剩够一个月运转的钱。但 15 年后，他的火箭回收了 200 多次。"
```

### 类型 4：代入型

```
❌ 普通开头："网络安全很重要。"
✅ 代入型："你手机里有 47 个 App 在收集你的位置信息。
其中 31 个，跟你用它完全没关系。"
```

### 类型 5：对话/问答型

```
"你知道世界上最贵的液体是什么吗？

不是石油，不是香水。

是人造的某种东西，每升价格超过 50 万美元。

猜猜是什么？"
```

## 结尾互动设计

### 互动方式

**1. 选择式**（评论率最高）
```
你觉得 AI 未来会：
A. 帮人类解决大问题
B. 制造更多麻烦
C. 不好说

评论区选一个，看看大家怎么选 👇
```

**2. 开放式提问**
```
如果你是 AI，你最想对人类说什么？

我先来——"你们写的 prompt 真的太烂了。"

你呢？评论区见 😂
```

**3. 共鸣式**
```
点赞收藏，下次朋友问"AI 到底是什么"，
直接把这篇甩给他。

然后深藏功与名 😎
```

**4. 预测式**
```
我赌一个赞：3 年内，"会用 AI"会像"会用电脑"一样成为基本技能。

同意的点个赞，不同意的评论区说服我 🔥
```

## Emoji 使用节奏

- 标题：一般不用 emoji（保持干净）
- 开头：可以有 1 个 🤯 / 💡 / ⚡
- 正文：每 3-4 段一个 emoji，做节奏分隔
- 结尾：1-2 个引导互动的 emoji 👇 🔥

**不要**：满篇 emoji，影响阅读
**可以**：关键节点用 emoji 做视觉路标
